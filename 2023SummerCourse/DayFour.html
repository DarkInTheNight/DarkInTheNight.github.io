Electricity

In today's lesson, the class learned about electrons, how they flow to create electric current, and about conductors and insulators. Using their new knowledge, they created electrical circuits using various foodstuffs as power supplies to power a clock in order to find out which ones were conductors and which ones were insulators for a scientific experiment. Some children even figured out how to power the clock using their own bodies!
